# Advent of Code 2025

This repository contains my solutions for Advent of Code 2025. I'll mainly be using Python here, with maybe a dash of C added in.

## Structure

There is a directory for each day named day_1 to day_12. Each one will contain an `input.txt` file and 2 files `p1` and `p2` for each part.

## About Advent of Code

Advent of Code is an Advent calendar of small programming puzzles created by **Eric Wastl**. These puzzles are for a variety of skill levels that can be solved in any programming language you like.

You can learn more at the official [Advent of Code](https://adventofcode.com/) website.

**Note:** This repository only contains my solutions for the challenge. To participate in it yourself, visit the website and login to access the problems.
